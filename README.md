# MySQL-journey
This repository documents my MySQL learning journey, covering database fundamentals, SQL syntax, and hands-on practice. It includes beginner-friendly notes, examples, and exercises on databases, tables, CRUD operations, joins, constraints, functions, and more to build a strong SQL foundation.
# Day 1 – Introduction to MySQL
Day 1 of my MySQL learning journey focuses on understanding the fundamentals of relational databases and MySQL. This file covers the concepts of databases, DBMS, RDBMS, MySQL architecture, SQL, tables, rows, columns, primary keys, and basic database operations. It also introduces the importance of databases in real-world applications, helping build a strong foundation for learning SQL and database management.
# Day2- Introduction
Day 2 of my MySQL learning journey focuses on the fundamentals of Database Management Systems. This file covers the differences between File Management Systems (FMS) and DBMS, basic database concepts such as data, fields, records, databases, DBMS, client-server architecture, protocols, and various data models with simple explanations and beginner-friendly examples.
# Day3-Indentifiers and Commands
Day 3 of my MySQL learning journey focuses on the fundamentals of SQL and database management. This file covers SQL identifiers and their naming rules, introduces SQL and its purpose, explains the different types of SQL commands (DDL, DML, DCL, TCL, and DQL), demonstrates database and table creation syntax, and provides an introduction to SQL data types with simple explanations and beginner-friendly examples.
# Day4-Alter Commads and Practice
Day 4 of my MySQL learning journey focuses on practicing the `ALTER TABLE` command through hands-on SQL exercises. This file covers creating tables, adding, modifying, renaming, and dropping columns, changing data types, renaming tables, and verifying schema changes using `DESC` and `SHOW TABLES`. It also includes a practical `COMPANY_STAFF` table exercise to strengthen MySQL table management skills.
# Day5-Insert Commands And Practice
Day 5 of my MySQL learning journey focuses on practicing DML (Data Manipulation Language) commands in MySQL. This file covers `INSERT`, `UPDATE`, `DELETE`, `TRUNCATE`, and `DROP` operations through hands-on examples using an employee table. It demonstrates different ways to insert records, update existing data, delete specific rows, manage table contents, and verify results with SQL queries, helping build a strong foundation in MySQL database manipulation. 
# Day6- MySQL Database Operations
Day 6: MySQL Database Operations and DCL Commands – Completed hands-on practice with table creation, data insertion, updating records, backup table creation, CRUD operations, and Data Control Language (DCL) commands, including user privilege management using GRANT and REVOKE. This session strengthened my understanding of MySQL database administration and security
# Day7-MySQL Transaction Control Language (TCL) & Locking
Day 7 learning SQL Focused on MySQL Transaction Control Language (TCL) and database concurrency concepts. Practiced COMMIT, ROLLBACK, and SAVEPOINT to manage transactions, explored ACID properties with real-world examples, and implemented table-level and row-level locking. Also learned different lock types, including Shared Lock, Exclusive Lock, and Intent Lock, to understand how MySQL maintains data consistency and integrity in multi-user environments
# Day7-MySQL Constraints, Referential Actions & SQL Operators
Implemented and practiced MySQL Constraints to maintain data integrity and consistency. Explored NOT NULL, UNIQUE, PRIMARY KEY, COMPOSITE KEY, FOREIGN KEY, CHECK, and DEFAULT constraints. Learned Referential Actions such as ON DELETE CASCADE, ON UPDATE CASCADE, ON DELETE SET NULL, and ON UPDATE SET NULL. Also performed SQL operator-based queries to calculate taxes, discounts, inventory values, unit pricing, and arithmetic operations on product data
